# Анализ финансовой устойчивости строительных МСП России (2021-2025)

## 📋 Описание проекта

Проект посвящен анализу финансовой устойчивости и рентабельности малых и средних предприятий (МСП) в строительной отрасли России в период макроэкономических шоков 2021-2025 годов.

## 🎯 Цель исследования

Оценка влияния макроэкономических факторов (пандемия, санкции, изменение ипотечных программ, ужесточение ДКП) на финансовое состояние строительных МСП.

## 📊 Источники данных

- **Реестр МСП** - данные о компаниях из Единого реестра субъектов малого и среднего предпринимательства.
- **Финансовая отчетность ФНС** - данные о прибыли (EBIT), дебиторской и кредиторской задолженности за 2021-2025 гг.
- **Объем выборки**: около 8 тыс. МСП.

## 🔍 Методология

1. **Сбор данных**: Парсинг финансовой отчетности с сайта Федеральной налоговой службы
1. **Предобработка**: Очистка реестра МСП, фильтрация строительных компаний
1. **Анализ**: 
   - Использование медианных значений для устойчивости к выбросам
   - Сегментация по размеру бизнеса (малые vs средние предприятия)
   - Географическая стратификация по регионам
   - Анализ динамики показателей за 5 лет
1. **Описательная статистика и визуализация**
1. **Формулирование бизнес-гипотез**
1. **Рекомендации для стейкхолдеров**

## 📈 Ключевые выводы

### Динамика отрасли
- **2025 г.**: медианная EBIT упала до минимума за 5 лет
- **Доля убыточных компаний** выросла с ~18% (2023) до ~25% (2025)
- **50% компаний** сократили прибыль с 2021 года
- **Средние предприятия** потеряли ~24% рентабельности с 2021 года

### Региональные особенности
**Некоторые лидеры по медианной EBIT:**
1. **ДНР** - масштабные госзаказы на восстановление инфраструктуры
1. **Ямало-Ненецкий АО** - нефтегазовое строительство
1. **Республика Саха** - добыча полезных ископаемых

### Сегментация по размеру бизнеса
- **Малые предприятия**: бурный рост в 2021-2024 гг. благодаря буму ИЖС по льготной ипотеке
- **Средние предприятия**: плавное снижение медианы из-за длительного строительного цикла и конкуренции с крупным бизнесом

### Распределение роста EBIT (2021-2025)
- **~25% компаний** стали убыточными
- **~50% компаний** снизили прибыль
- **Длинный правый хвост** (до +520%) - компании в регионах с госзаказом

## 🚀 Как запустить

1. Клонируйте репозиторий:
```bash
git clone <https://github.com/xTravisBickley/construction-sme-eda.git>
cd construction-sme-eda
```

1. Установите зависимости:
```bash
pip install pandas numpy matplotlib jupyter
```

1. Запустите Jupyter Notebook:
```bash
jupyter notebook
```

1. Откройте ноутбуки в порядке выполнения:
   - `clearing_registry.ipynb`
   - `parsing_fts.ipynb`
